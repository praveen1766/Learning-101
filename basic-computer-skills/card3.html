Card 3: Using Mouse and Keyboard

In Andhra Pradesh, just like using a bicycle to go to the market, the mouse and keyboard are your tools to control the computer. Learn how to click, type, and use shortcuts to make your work faster, like a skilled farmer using the right tools for farming.

Using the Mouse

Move the mouse to point the cursor on the screen. Click once to select, double-click to open, right-click for options, and drag to move items. It's like pointing with your finger in a book.

Using the Keyboard

Type letters, numbers, and symbols by pressing keys. Use Shift for capital letters, Space for gaps, and Enter to start a new line. Special keys like Backspace delete mistakes.

Basic Shortcuts

Ctrl+C to copy, Ctrl+V to paste, Ctrl+Z to undo. These save time, like using a tractor instead of a plow. Practice them to work faster.

Key Terms

  • Mouse: A device to point and click on the screen.
  • Keyboard: Keys to type letters and commands.
  • Click: Pressing the mouse button once.
  • Double-click: Pressing twice quickly to open.
  • Shortcut: Key combinations for quick actions.
