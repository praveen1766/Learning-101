Card 2: Storing and Retrieving Data

Now that you know what a database is, let's learn how to add and find information. Like organizing ration cards in a village office.

Adding Data

Story: Insert new records, like adding a family's details to the ration database.

Fun Fact: In Andhra Pradesh, PDS adds new beneficiaries regularly.

Finding Data

Story: Use queries to search. "Show families in Vijayawada with 4 members."

Fun Fact: Farmers query crop data to plan sales.

Updating Data

Story: Change info, like updating a phone number when someone moves.

Fun Fact: Banks update balances instantly in databases.

Key Technical Terms

  • Insert: Adding new data to the database.
  • Query: A request to find specific data.
  • Update: Changing existing data.
  • Delete: Removing data (use carefully).
  • SQL: Language for talking to databases, like "SELECT * FROM families".
