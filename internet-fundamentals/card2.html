Card 2: How the Internet Works

The internet is like a huge network of roads connecting villages. In Andhra Pradesh, it's like the highways linking Hyderabad to Tirupati – data travels fast!

Connecting Devices

Story: Your phone connects via WiFi or mobile data to the internet.

Fun Fact: In Andhra Pradesh, even remote areas get internet via towers.

Data Travels

Story: Information goes through cables and satellites to reach websites.

Fun Fact: Undersea cables connect India to the world, like rivers.

Websites and Servers

Story: Websites are stored on servers. You request a page, server sends it back.

Fun Fact: Andhra Pradesh government sites run on servers in Hyderabad.

Key Technical Terms

  • WiFi: Wireless way to connect to the internet.
  • Browser: App like Chrome to view websites.
  • URL: Address of a website, like https://example.com.
  • Bandwidth: How fast data travels, like road width.
  • IP Address: Unique number for each device on the internet.
