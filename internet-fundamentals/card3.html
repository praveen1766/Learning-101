Card 3: What is a Website?

A website is like a house built for the internet. Imagine building a new house in a village near Visakhapatnam. It has a structure, it has decorations, and it has working things like lights and fans. Websites are very similar!

HTML: The Structure

Story: HTML is like the bricks, cement, and layout of the house. It defines the rooms, the doors, and the windows. It gives the website its basic shape.

Analogy: The basic plan of the house is the HTML.

CSS: The Decoration

Story: CSS is like painting the walls, choosing the color of the curtains, and arranging the furniture. It makes the house look beautiful and stylish.

Analogy: The beautiful rangoli at the entrance is the CSS.

JavaScript: The Interactivity

Story: JavaScript is like the electrical wiring. It makes things work! When you flip a switch, the light turns on. On a website, when you click a button, something happens.

Analogy: The switch for the fan is the JavaScript.

Key Technical Terms

  • HTML (HyperText Markup Language): The standard language for creating web pages.
  • CSS (Cascading Style Sheets): The language for styling the visual presentation of a web page.
  • JavaScript: A programming language that enables interactive web pages.
  • Frontend: What you see and interact with in your browser (the house). Made with HTML, CSS, and JS.
  • Backend: The server, database, and application logic that you don't see (the power station and water supply).
